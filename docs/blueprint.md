# **App Name**: StyleAI

## Core Features:

- Image Upload: Allow users to upload images of their clothing items.
- Wardrobe Display: Display all uploaded clothing items in a user's personal wardrobe.
- AI Styling Assistant: Chat with an AI tool to receive outfit suggestions based on the user's wardrobe and needs.

## Style Guidelines:

- Primary color: Muted violet (#9B8FCF) for a sophisticated, calming feel.
- Background color: Very light gray (#F0F0F5), close to white, to create a clean and modern backdrop that ensures excellent contrast.
- Accent color: Soft blue (#77B5FE) for interactive elements and highlights, adding a touch of vibrancy without overwhelming the palette.
- Body and headline font: 'Be Vietnam Pro', a versatile sans-serif known for its clarity, ensuring an accessible and user-friendly reading experience.
- Use clean, minimalist icons to represent different clothing categories and actions.
- Employ a clean and organized layout, making it easy for users to browse and manage their wardrobe items.
- Subtle transitions and animations to provide feedback during image uploads and AI styling suggestions.
